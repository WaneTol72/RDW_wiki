Прошивка полей RDW A520 UEFI Shell

Директория:​

![[Pasted image 20250127174233.png]]

Что нужно сделать:​

  1. Загрузиться с нужной флешки в UEFI Shell
  2. Выбрать нужную флешку (map в помощь):
fs0:
  1. Создать переменную a:
  • Написать:
set a=
  • Пикнуть или ввести серийный номер готовой продукции RDW
  1. Запустить скрипт flash.nsh:
flash.nsh
  1. Смотреть и ждать. Потом проверить BIOS

Опционально:​

  • Скрипт flash.nsh:
@echo -off
echo "Serial Number:"
set a

amideefix64.efi /SS %a%

amideefix64.efi /BP "RDW A520"
amideefix64.efi /SM "RDW Computers"
amideefix64.efi /SP "RDW Kama-06"
amideefix64.efi /CT "0D"

echo "Done"

pause

[!WARNING] Перед запуском, нужно сменить поля в этом файле на нужные! Где, /CT - тип устройства (mini PC, All in One, Desctop Case, Notepad)
