# Активация Windows
---
> ***Использование KMS-активации через командную строку с помощью 25-значного ключа продукта***
#### CMD/Командная строка

- Открыть командную строку от имени администратора.
	1. открыть *Пуск меню*
	2. в поиске ввести **cmd** 
	3. выбрать интересующий нас вариант
- В командной строке ввести следующий набор скриптов.
	```cmd
	slmgr /ipk[ключ]
	```
	затем
	```cmd
	slmgr /ato
	```
	скрипт ***slmgr /ipk{ключ}*** - заменяет ключ на тот что был введен
	скрипт ***slmgr /ato*** - проверяет активацию принудительно (если все хорошо, система покажет окно об успешной активации продукта)
---

> ***Активация с использованием встроенного функционала***

**Использование ключ продукта**:
 *Параметры* > *Система* > *Активация* > *Изменить ключ продукта* и
 ввести 25-значный код.
