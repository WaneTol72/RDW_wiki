# Активация Windows
---
> **Активация с помощью 25-значного ключа продукта с использованием командной строки**
#### CMD/Командная строка

- Открыть командную строку от имени администратора.
	1. открыть *Пуск меню*
	2. в поиске ввести **cmd** 
	3. выбрать интересующий вариант
- В командной строке ввести следующий набор скриптов.
	```cmd
	slmgr /ipk[ключ]
	```
	затем
	```cmd
	slmgr /ato
	```
