# Алгоритм работы с видеокартой

-  ***CDI*** - Установить ПО для устройства.
	если после установки драйвера устройство не работает
-  ***Internet*** - скачать драйвер с официального сайта [NVidia](https://www.nvidia.com/en-us/drivers/) ,
	[AMD Radeon](https://www.amd.com/en/support/download/drivers.html) ,
