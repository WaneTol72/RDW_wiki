# Установка клиентской ОС/Windows:

-   **WDS**  - при условии что в ***BIOS***, включен _Boot from onboard LAN_ (_UEFI: PXE_).
	1. Прогружаемся (*исключение: для некоторых MB нужно через boot menu выбрать протокол PXE IP4*) до окна выбора ОС.
	2. Находим нужную нам ОС , ***согласно ТЗ*** (версия ОС, № заказа/подзаказа) устанавливаем. *Исключение: если в комплектации более одного накопителя (**HDD/SSD/m.2**)*
