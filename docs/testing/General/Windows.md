# Установка клиентской ОС/Windows:

-   **WDS**  - при условии что в ***BIOS***, включен _Boot from onboard LAN_ (_UEFI: PXE_).
	1. Прогружаемся (*исключение: для некоторых MB нужно через boot menu выбрать протокол PXE IP4*) до окна выбора ОС.
	2. Выбираем интересующую нас ОС  согласно ТЗ  (версия ОС, № заказа/подзаказа). *Исключение: если в комплектации более одного накопителя (**HDD/SSD/m.2**)*
	3. Выбор накопителя производится согласно ТЗ, либо выбирается более скоростной (m.2 -> SSD -> HDD). В случае когда установлены два одинаковых накопителя, выбираем по номеру порта подключения (port_0/SATA_0).
