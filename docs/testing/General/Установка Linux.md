# Установка Linux
---

> [!WARNING] ***Внимание*** 
> *Предварительно (при необходимости) сделать все необходимые настройки BIOS согласно ТЗ.*

>Включение протокола *Boot from onboard LAN (UEFI: PXE IPv4)/Network stack/UEFI PXE Support* для установки Linux не требуется.

---

Установку ***Linux*** производим через флешку.

- Вставить флешку в выключенное изделие. Включить изделие.
- Вызвать *Boot menu* (клавиша вызова в соответствии с моделью MB), выбрать в списке флешку.
- После загрузки появиться меню выборки (нас интересует вариант №3)
